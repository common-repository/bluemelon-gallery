This plugin provides an easy and efficient way how to connect with your bluemelon gallery.

== Description ==
This plugin was develeped for http://www.bluemelon.com users who want to present their photos directly in WordPress. You can insert a fully working gallery into your blogs. Another big advantage of this plugin is the ease of inserting a gallery. It is as simple as clicking an icon placed in your wordpress editor, inserting a full link to your bluemelon album or category you want to show in your blog, choosing the height of gallery and clicking the Insert button. Very easy and intuitive interface. Bluemelon also provides the possibility of customization of gallery appearance. At http://www.bluemelon.com/profile/wordpress you can create a look which will best fit your blog.


== Installation ==
1. Unzip the bluemelongallery.zip file.
2. Create directory "bluemelon-gallery" in wp-contents/plugins.
3. Upload the unziped files to bluemelongallery folder you recently created in 2. step ( wp-contents/plugins/bluemelon-gallery ). If you're using FTP, use 'binary' mode.
